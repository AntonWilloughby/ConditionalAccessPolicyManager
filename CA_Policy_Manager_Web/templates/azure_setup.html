Azure App Registration Setup

Automatic Setup

This will automatically create an Azure App Registration with the required permissions.

Prerequisites:
  • Azure CLI installed (Download)
  • Azure account with permissions to create app registrations
  • PowerShell available on your system

Setting up Azure App Registration... This may take 1-2 minutes.

Manual Setup

Prefer to set up manually? Follow these steps:

  1. Go to Azure Portal - App Registrations
  2. Click New registration
  3. Set name: CA Policy Manager
  4. Set redirect URI: http://localhost:5000/auth/callback
  5. Add API permissions:
    • Policy.Read.All
    • Policy.ReadWrite.ConditionalAccess
    • Directory.Read.All
    • Group.ReadWrite.All
  6. Create a client secret
  7. Update .env file with Client ID and Secret
  8. Set DEMO_MODE=false in .env
  9. Restart the Flask app
